# Product Sense: EV 폐배터리 실시간 관제 클라우드 전환 PoC

## 1. Background (배경)
- 의 전기차(EV) 폐배터리 회수 및 재활용 물류 사업 확장.
- 이동 중인 폐배터리는 화재 위험이 있어 1초 단위의 실시간 상태(온도, 전압, GPS 등) 모니터링이 필수적임.
- 기존 온프레미스 기반의 레거시 통합 물류 시스템은 대규모 실시간 IoT 텔레메트리 데이터를 감당할 수 없어 시스템 장애 위험이 존재함.

## 2. Objectives (목표)
- 레거시 시스템 전체를 수정하는 대신, '실시간 배터리 트래킹' 모듈만 분리하여 클라우드 네이티브(MSA) 환경으로 구축하는 PoC(사전 검증) 수행.
- 대규모 트래픽 발생 시 클라우드의 자동 확장(Auto-scaling) 능력을 검증하여, 향후 글로벌 물류 시스템의 IT 인프라 고도화 방향성을 제시.

## 3. Core Features (핵심 기능)
- **IoT Data Ingestion:** 초당 수만 건의 배터리 센서 데이터를 병목 없이 수집 및 큐잉.
- **Real-time Alerting:** 배터리 온도가 임계치를 초과할 경우 즉각적인 알림(Alert) 이벤트 발생.
- **Legacy Integration:** 클라우드에서 정제/처리된 중요 데이터(예: 배터리 회수 완료 상태)만 기존 사내 레거시 시스템으로 주기적으로 동기화.

## 4. Success Metrics (성공 지표)
- **가용성/성능:** 부하 테스트(Load Testing) 시 10,000 TPS(초당 트랜잭션) 환경에서 p95 응답 지연 시간(Latency) 500ms 미만 유지.
- **확장성:** 트래픽 폭증 시 인프라(컨테이너)가 자동으로 Scale-out 되는지 검증.
